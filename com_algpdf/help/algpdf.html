ALGPDF - Analyze Links and Generate PDF

This component, given an newsletter, analyze all the links, associated to contents and attachments,
and generates the complete pdf, with all information about these newsletter.

 

This component has thee parts:
  • Analyze Links & Generate PDF
  • Listing of the generated PDFs
  • Configure

Analyze Links & Generate PDF

Step 1 - Select Content Item to Analyze Links

This form has the following fields:
  • Contents - Listbox to specify what content or newsletter is to analyze links.
    This listbox items can be modified, changing the categories to analyze in configure interface
  • Name of the pdf - Indicates the name of pdf to generated.
    When select an content, it automaticaly pad this field.
    This field can contains only letters ([a-z]), numbers ([0-9]) and underscore([_]).
  • Overwrite pdf ? - This fied indicates if you want overwrite this pdf file, specified in the previows field.
When filled these fields, click in the "Analyse Link" button.

Step 2 - Analyzing Links

In this interface, given the content selected, is analyzed all the links of there (contents and attachements).
During the process, its showed a mode verbose, where you can see what it is making.
The "continue" button is only enable, after end processing.
When processing is ended, click in the "continue" button.

Step 3 - Listing of the detected links

In this interface, is showed two listings: one to attachements and other to contents.
Now, you may specify what items you want append to pdf, but you must have at least one item checked.
Tip: If you want select all or none items, check or uncheck the first checkbox, in the headings of each listing.
When finished selection, click in the "Generate PDF" button.

Step 4 - Generating PDF

In this interface is generated the pdf.
During the process, its showed a mode verbose, where you can see what it is making.
The "finish" button is only enable, after end processing.
When finished, click in the "finish" button, and it goes to the listing of the pdfs, where you can see the pdfs generated.

Listing of the generated PDFs

In this interface, is showed a listing of the generated pdfs.
It list a several properties of the generated pdfs, including the initial content selected.
Clicking in the file name, you can view the pdf, in another window.
To remove items, check these items, and click in the "Remove" icon in the toolbar.
Tip: If you want select all or none items to remove, check or uncheck the first checkbox, in the headings.

Configure /Configuration

In this form, you can customize the settings of this component.
This form has the following fields:
  • Recursive - It indicates if you want analyse only all the links of these newsletter (no), or if you want analyze all the links of these newsletter and the links in its related contents (yes).
  • Verify contents - This field indicates if you want append the related contents to pdf.
  • Verify attachs - This field indicates if you want append the related attachements to pdf.
  • Show categorie - This field indicates if you want append the categories titles, between contens.
  • Path for storing file - Indicates the path where you want store pdfs.
    It must be equal of the path in Docman component (configurations).
  • Categories to analyze - Indicates what categories you want to list in the step 1 of Analyze Links and Generate PDF.
